# Technical Decisions

## State Management

Used React Context API because the application is small and does not require Redux.

## Styling

Used SCSS Modules to keep styles scoped and maintainable.

## Data Fetching

Used Fake Store API as required by the assignment.

## Persistence

Used localStorage to persist cart state across page refreshes.

## Trade-offs

Did not implement backend persistence or authentication to keep the scope focused on frontend functionality.
